# 🍴 Food Court - Shopping Cart

A simple and responsive **Food Shopping Cart** web application built using **HTML, CSS, and JavaScript**. Users can browse food items, add products to the cart, manage quantities, and view the total price.

## 🚀 Live Demo

https://sankari-d27.github.io/Shopping-Cart/

## 📌 Features

- 🛒 Add food items to cart
- ➕ Increase product quantity
- ➖ Decrease product quantity
- 🗑️ Remove items from cart
- 💰 Automatic total price calculation
- 🔢 Cart item count display
- 📱 Responsive design for different screen sizes
- 📂 Products loaded dynamically from JSON data
- 🖼️ Dynamic product images

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- JSON
- Font Awesome Icons
- Google Fonts
